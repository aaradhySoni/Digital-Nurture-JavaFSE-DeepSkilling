# Week 5 — React Hands-on Lab Assignments

This README documents all 10 exercises completed for the Week 5 React assignment.
Each exercise is an **independent** Create React App (CRA) project — it has its own
`package.json` and can be run on its own without depending on any other exercise folder.



## How to run any exercise

```bash
cd "exercise <number>"
npm install
npm start
```

Then open `http://localhost:3000` in your browser.
