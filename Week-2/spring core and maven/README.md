# Spring Core and Maven

This folder contains separate Maven projects for the Spring Core and Maven exercises.

## Exercise 1

Creates a basic Spring XML application context and defines `BookService` and `BookRepository` beans.

## Exercise 2

Updates the XML configuration to inject `BookRepository` into `BookService` using setter-based dependency injection.

## Exercise 4

Creates a Maven project configured with Spring Context, Spring AOP, Spring WebMVC, and the Maven Compiler Plugin for Java 1.8.
